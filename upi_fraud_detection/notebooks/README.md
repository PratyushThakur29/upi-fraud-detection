# Jupyter Notebooks

This directory contains Jupyter notebooks for:

1. **Exploratory Data Analysis** - Data exploration and visualization
2. **Model Experiments** - Testing different model configurations
3. **Feature Engineering** - Feature importance and engineering experiments
4. **Results Analysis** - Analyzing model predictions and errors

## Usage

```bash
# Install Jupyter
pip install jupyter

# Start Jupyter
jupyter notebook

# Create new notebook or open existing ones
```
